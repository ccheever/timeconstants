timeconstants
=============

Time constants for JavaScript, ex. hour, minute, second, etc.

## Usage

```js
var constants = require('timeconstants');

constants.ms;
constants.second;
constants.minute;
constants.hour;
constants.day;
constants.week;

// = 1/12th of a year
constants.monthApprox;

// = 365.25 days
constants.yearApprox;
```
